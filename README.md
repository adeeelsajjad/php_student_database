🎓 Student Management System

A simple and efficient Student Management System built using PHP and MySQL, running on the Laragon local development environment.
This project allows users to manage student records with full CRUD functionality.

📌 Features

➕ Add New Students

📋 View All Students

✏️ Edit Student Details

❌ Delete Student Records

🔄 Auto ID Management

🏙️ City Field Support

💾 Data Stored in MySQL Database

🛠️ Technologies Used

PHP – Backend Development

MySQL – Database

HTML5 & CSS3 – Frontend Design

Laragon – Local Server Environment

📂 Project Structure
Student-Management-System/
│
├── index.php          # Main dashboard page
├── add.php            # Add new student
├── edit.php           # Edit student details
├── delete.php         # Delete student
├── DB.php             # Database connection file
├── style.css          # Styling file
└── README.md          # Project documentation
⚙️ Installation & Setup (Using Laragon)

Install Laragon from: https://laragon.org

Move the project folder into:

C:\laragon\www\

Start Laragon.

Open phpMyAdmin from Laragon.

Create a new database (example: student_db).

Import the provided SQL file (if available).

Update database credentials inside DB.php:

$host = "localhost";
$user = "root";
$password = "";
$database = "student_db";

Open browser and run:

http://localhost/Student-Management-System/
